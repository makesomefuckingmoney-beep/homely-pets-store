# Homely Pets — Master packaging scheme (supplier)

Use this for **every** product label. Match the Adult Dog · Chicken & Sweet Potato reference exactly.

## Colours (print CMYK from these RGB references)

| Role | Hex | Use |
|------|-----|-----|
| Cream background | `#F5F0E6` | Main panel |
| Forest green | `#2D4A32` | Headlines, logo text, bottom band |
| Sage green | `#3F5C46` | Secondary text |
| Gold accent | `#C4A052` | Lines, badge trim, tagline, icons |
| Tan logo trees | `#B8956B` | House + pine tree icon |
| Bottom band | `#2D4A32` | Feature icon row (same as forest) |

## Layout (fixed — do not change)

1. **Top:** House + 6 pine trees icon (tan/gold) · `Homely Pets™` · ♥ `NOURISHING COMPANIONS, NATURALLY`
2. **Centre left:** Large life-stage line (e.g. `Adult Dog`) · Flavour line · Badge with wheat icons
3. **Centre right:** Pencil-sketch pet + pine trees + mountain
4. **Bottom curve:** Gold separator · 4 icon columns with short claims · `NET WEIGHT` + weight (e.g. `2kg`)

## Typography

- Headlines: high-contrast serif (Cormorant Garamond or equivalent)
- Legal / icons: small caps sans
